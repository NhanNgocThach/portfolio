Single-page portfolio website for the INTO Suffolk Integrated Master's Program.

## How to publish on GitHub Pages

1. Create a new repository on GitHub (public). A common name is `portfolio` or `<your-username>.github.io`.
2. Upload this entire folder's contents (`index.html`, `README.md`, `assets/`) to that repo.
3. In the repo, go to **Settings → Pages**.
4. Under **Source**, pick **Deploy from a branch**, select `main` and `/ (root)`, then **Save**.
5. After a minute, your site is live at `https://<your-username>.github.io/<repo-name>/`.
